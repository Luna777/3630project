Variable

Sample code:
public class MyProgram{
    public static void main(String[] args){
        String s1 = "HELLO";
        int numb1 = 10;

        System.out.println( s1 );
        System.out.println( numb1 );
    }
}
Output:
HELLO
10
Now we have a more complicated program. Inside main method, the first and second line are the examples of variable.

"s1" is the name of first variable, "String" is the data type of "s1" and it means a string of characters which is words or sentences. "String s1" is the declare of variable "s1", and then, we use "=" sign to assign the value "HELLO" to s1. By the end of this line, we use ";" to end this action of declaration and assign of the first variable "s1".

The second line "int numb1 = 0;" does the same thing as "s1", only here we are making an integer number variable called "numb1" and give it a value of "10".

Why we want to use variables to store values? For expression, calculation, etc, we can use them multiple times or involve them in other functions calculation.

We can see variables as objects in a house, for example, an apple. We might not use the apple immediately, but it is there and later on, we could use it to make apple juice or apple pie.
