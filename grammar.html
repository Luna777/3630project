Grammar

Sample code:
public class MyProgram{
    public static void main(String[] args){
        String s1 = "HELLO";
        int numb1 = 10;

        System.out.println( s1 );
        System.out.println( numb1 );
    }
}

From all the examples we had seen so far, we saw grammar everywhere. Grammar in java is like the grammar in other languages, it is a way to combine words and making the language is understandable by computer. Ordering, syntax, punctuations, marks are all grammar in Java. For example, we always put ";" at the end of a statement; we use "{ }" to hold on the contents of class and method. we use "=" to assign values.

Grammar puts words together to make them to be Java language. It is like cement of houses, like the ordering or furniture. It is everywhere in Java.

In the future, we are learning more and more Java knowledge, then we are going to see more grammars.
