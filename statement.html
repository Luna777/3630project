Statement

Sample code:
public class MyProgram{
    public static void main(String[] args){
        String s1 = "HELLO";
        int numb1 = 10;
        int numb2 = 5;
        int numb3 = numb1 + numb2;

        System.out.println( s1 );
        System.out.println( numb1 );
        System.out.println( numb2 );
        System.out.println( numb3 );
    }
}
Output:
HELLO
10
5
15

We just learned how to make variables and assign values to them. The line or the sentence we just done is a statement. "String s1 = "HELLO"; " is a statement; "int numb1 = 10; " is another statement. Each statement ends by a ";" sign. "int numb3 = numb1 + numb2;" is assigning the sum of numb1 and numb2 to numb3, so numb3 is going to be 15.

Also in the sample code, "System.out.println( S1 );" and "System.out.println( number );" are both statements. They are making system calls to make computer output what is in its parentheses. In this case, these two statement output the values of s1 and numb1.

From those examples, the functions of statement are creating variables, assignment of values, making function/method calls. Therefore, we can see statement is an action. Forexample, "System.out.println( )" is like a person saying something, and "s1" in the parentheses is the word that going to say.
