Class

Sample code:
public class MyProgram{
     public static void main(String[] args){
         System.out.println("HELLO!");
     }
}
This is the first program we are going to look at.
The first line shows our class is called MyProgram. Followed the name is a { sign and by the end of a class, there is a } sign.
This is a complete class in Java:

public class MyProgram{

}

Inside a class, we can build methods to make computer do things for us. We can see a class as a frame of a house. Everything inside a class is the content of house.
